Reinstall_ubuntu

2020-01-28

旧系统该保存什么

/home下的所有东西 /root下看看有没有东西 /root/.bashrc /etc/profile可以保留一份。

安装

  1. 用软碟通制作U盘系统
    1. 用软碟通打开.iso文件
    2. “写入硬盘镜像”，写入方式默认HDD+，如果发现制作好了进不去系统，可以选择raw。
  2. U盘启动
  3. 看情况选择安装选项
  4. 安装完成之后U盘可能会显示只有十几K的剩余空间了不能用了，这个时候可以用重新挂载的方式清除U盘里面的系统，变成一个空U盘。
    • Win+R运行diskpart
    • list disk
    • select disk [U盘的序号]
    • clean
    • 打开计算机->管理->磁盘管理,给U盘新建一个简单卷。

  • 添加root

  • 换软件源
  • 更新语言包

中文输入法

安装搜狗拼音输入法和fcitx。 将输入法设置为fcitx，重启机器，之后在fxitx的配置中添加搜狗拼音输入法。 使用shift更换中英文。
